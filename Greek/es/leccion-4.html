Vocales y diptongos

Las 7 vocales

α (a), ε (e corta), η (e larga), ι (i), ο (o corta), υ (u/y), ω (o larga). Cantidad: largas: η, ω, α larga, ι larga, υ larga; cortas: ε, ο, α corta.

Diptongos propios (8)

αι (ai)
ει (ei)
οι (oi)
υι (yi)
αυ (au)
ευ (eu)
ου (ou)
ηυ (ēu)

Diptongos impropios (iota suscrita)

ᾳ (ᾱι), ῃ (ηι), ῳ (ωι) — se pronuncian como α larga, η, ω respectivamente. Ejemplo: τῇ (a la, dativo).

αι = "ai" como en "aire"; ει = "ei" como en "reina"; οι = "oi" como en "oiga"; υι = "ui"; αυ = "au" como "aula"; ευ = "eu" como "Europa"; ου = "ou" como "tour"; ηυ = "eu" alargado.
Contracciones vocálicas básicas

En griego, cuando dos vocales se encuentran, a menudo se contraen. Reglas principales:

  • α + ε = α (ej. τιμάετε → τιμᾶτε)
  • α + ο = ω (ej. τιμάομεν → τιμῶμεν)
  • ε + ε = ει (ej. φιλέετε → φιλεῖτε)
  • ε + ο = ου (ej. φιλέομεν → φιλοῦμεν)
  • ο + ε = ου (ej. δηλόετε → δηλοῦτε)
  • ο + ο = ου (ej. δηλόομεν → δηλοῦμεν)
Práctica: ¿cómo se contrae ποιέω (hacer) en presente?
ποιέω → ποιῶ (por contracción ε+ω = ω). Tercera persona singular: ποιέει → ποιεῖ.
Cambios consonánticos

Cuando un verbo o sustantivo termina en consonante y se añade σ (futuro, aoristo, etc.), ocurren cambios regulares:

  • Labiales (π, β, φ) + σ = ψ
    Ejemplo: γράφω → γράψω (escribiré)
  • Velares (κ, γ, χ) + σ = ξ
    Ejemplo: ἄγω → ἄξω (conduciré)
  • Dentales (τ, δ, θ) + σ = σ (la dental desaparece)
    Ejemplo: πείθω → πείσω (persuadiré)
¿Cómo se forma el futuro de λύω (desato)?
λύω → λύσω (no hay cambio porque la raíz termina en vocal).
