1. ¿Qué son las "Seis Formas Principales"?

Las Seis Formas Principales (en inglés Principal Parts) son las seis formas verbales básicas que, una vez memorizadas, permiten deducir todas las demás formas de un verbo griego.

Las seis formas son:
  1. Presente activo/medio (1ª sg.) – base para presente, imperfecto y subjuntivo presente.
  2. Futuro activo/medio – base para el futuro.
  3. Aoristo activo/medio – base para el aoristo activo y medio.
  4. Perfecto activo – base para el perfecto y pluscuamperfecto activos.
  5. Perfecto medio/pasivo – base para el perfecto y pluscuamperfecto medio/pasivos.
  6. Aoristo pasivo – base para el aoristo y futuro pasivos.

Importancia exegética: Conocer las formas principales de un verbo permite reconocer cualquier forma que aparezca en el texto, incluso si no se ha visto antes. Es la herramienta más poderosa para la lectura fluida del NT.

Ejemplo:
γίνομαι → γέγονα (perfecto activo) – si conoces la 4ª forma, reconoces que γέγονα es perfecto de γίνομαι.
λύω (desatar):
1. λύω (presente activo)
2. λύσω (futuro activo)
3. ἔλυσα (aoristo activo)
4. λέλυκα (perfecto activo)
5. λέλυμαι (perfecto medio/pasivo)
6. ἐλύθην (aoristo pasivo)
2. Información que provee cada forma

Cada una de las seis formas principales revela información crucial sobre el verbo:

FormaQué revelaEjemplo (λύω)
1. PresenteTema del presente; tipo de verbo (-ω, -μι, contracto)λύω
2. FuturoTema del futuro; cambios consonánticos ante σλύσω
3. Aoristo activo/medioTema del aoristo activo; aumento; forma del aoristo (1° o 2°)ἔλυσα
4. Perfecto activoTema del perfecto activo; reduplicación; vocal de la raízλέλυκα
5. Perfecto medio/pasivoTema del perfecto medio/pasivo; cambios consonánticosλέλυμαι
6. Aoristo pasivoTema del aoristo pasivo; si tiene -θ- o no (1° o 2° pasivo)ἐλύθην
Clave: Con estas seis formas, puedes generar todos los tiempos y modos del verbo: presente, imperfecto, futuro, aoristo, perfecto, pluscuamperfecto, subjuntivo, optativo, imperativo, infinitivo y participio.
Verbos irregulares cambian de raíz:
φέρω (llevar) → 2. οἴσω → 3. ἤνεγκα → 4. ἐνήνοχα → 5. ἐνήνεγμαι → 6. ἠνέχθην.
Cada forma revela una raíz diferente: φερ-, οἰσ-, ἐνεγκ-, ἐνηνοχ-, ἐνηνεγμ-, ἐνεχθ-.
3. Patrones de formación y cambios vocálicos

Los verbos griegos muestran patrones predecibles en sus formas principales, aunque hay muchas irregularidades que deben memorizarse.

Cambios vocálicos (ablaut) en la raíz

  • Grado ε (presente): λέγω
  • Grado ο (aoristo/perfecto): λέλεκα, λέλογα
  • Grado cero (participios): λεκτός
Ejemplo: γίνομαι (ε/ο en la raíz gen-/gon-/gn-).

Cambios consonánticos ante -σ-

  • Labiales (π, β, φ) + σ → ψ (ej. γράφω → γράψω)
  • Velares (κ, γ, χ) + σ → ξ (ej. ἄγω → ἄξω)
  • Dentales (τ, δ, θ) + σ → σ (ej. πείθω → πείσω)

Reduplicación en el perfecto

  • Silábica: repite la primera consonante + ε (ej. λύω → λέλυκα)
  • Temporal: alarga la vocal inicial (ej. ἀκούω → ἀκήκοα)
  • Con reduplicación en -μι: (ej. δίδωμι → δέδωκα)
Cambios ante -σ-:
π, β, φ → ψ · κ, γ, χ → ξ · τ, δ, θ → σ (o desaparece).
Reducción del aumento: ε- para verbos que empiezan por consonante; alargamiento de vocal para verbos que empiezan por vocal (ἀκούω → ἤκουσα).

5. Deducción de formas a partir de las principales

Una vez conocidas las seis formas principales, puedes deducir cualquier forma verbal.

Regla de deducción:
  • Presente/Imperfecto: usa la 1ª forma + terminaciones primarias/secundarias.
  • Futuro: usa la 2ª forma + terminaciones primarias (sin aumento).
  • Aoristo activo/medio: usa la 3ª forma (con aumento) + terminaciones secundarias.
  • Perfecto activo: usa la 4ª forma (con reduplicación) + terminaciones de perfecto.
  • Perfecto medio/pasivo: usa la 5ª forma + terminaciones de perfecto medio/pasivo.
  • Aoristo pasivo: usa la 6ª forma (con aumento y -θην) + terminaciones secundarias activas.

Ejemplo con γίνομαι:

  • 1ª: γίνομαι → presente medio (1ª sg.)
  • 2ª: γενήσομαι → futuro medio (1ª sg.)
  • 3ª: ἐγενόμην → aoristo medio (1ª sg.)
  • 4ª: γέγονα → perfecto activo (1ª sg.)
  • 5ª: γεγένημαι → perfecto medio/pasivo (1ª sg.)
  • 6ª: ἐγενήθην → aoristo pasivo (1ª sg.)
Ejercicio mental: Si ves γεγέννηται, sabes que es perfecto medio/pasivo de γεννάω porque la 5ª forma es γεγέννημαι.
ἐγράφη → aoristo pasivo de γράφω (6ª forma: ἐγράφην).
ἀκηκόαμεν → perfecto activo de ἀκούω (4ª forma: ἀκήκοα).
ἐλήλυθα → perfecto activo de ἔρχομαι (4ª forma: ἐλήλυθα).

12. Lectura programada · Romanos 3:22-26 y Juan 3:16

Lee los textos. Identifica la forma principal de cada verbo.

Romanos 3:22 – δικαιοσύνη

δικαιοσύνη δὲ θεοῦ διὰ πίστεως Ἰησοῦ Χριστοῦ εἰς πάντας τοὺς πιστεύοντας

δικαιόω – sus formas principales son: δικαιόω, δικαιώσω, ἐδικαίωσα, δεδικαίωκα, δεδικαίωμαι, ἐδικαιώθην.
Romanos 3:24 – δικαιούμενοι

δικαιούμενοι δωρεὰν τῇ αὐτοῦ χάριτι

Presente pasivo participio de δικαιόω – deriva de la 1ª forma (δικαιόω) con terminaciones medias/pasivas.
Juan 3:16 – ἠγάπησεν

οὕτως γὰρ ἠγάπησεν ὁ θεὸς τὸν κόσμον

3ª forma (aoristo activo): ἠγάπησα – 3ª persona singular ἠγάπησε(ν).
